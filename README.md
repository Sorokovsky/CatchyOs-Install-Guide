# Повна інструкція з встановлення CachyOS на ПК з i5-10400F, GTX 1650, SSD + HDD

Цей посібник створено на основі детальної консультації та враховує всі особливості вашого обладнання.

## 1. Підготовка до встановлення (на ПК з Windows)

Перед тим, як встановлювати CachyOS, виконайте наступне у вашій поточній системі Windows:

1.  **Зробіть резервну копію всіх важливих даних** із SSD та HDD на зовнішній носій. Це головне правило безпеки.
2.  **Вимкніть Secure Boot** у налаштуваннях BIOS/UEFI. Це необхідно для коректної роботи драйверів NVIDIA.
3.  **Вимкніть "Швидкий запуск" (Fast Startup)** у Windows. Це дозволить уникнути проблем з монтуванням дисків.
4.  **Завантажте ISO-образ CachyOS** з офіційного сайту.
5.  **Створіть завантажувальну флешку** за допомогою програми Rufus (в Windows) або balenaEtcher.

## 2. Оптимальна схема розмітки дисків

Ваша конфігурація: SSD 238 ГБ + HDD 1.82 ТБ. Ми будемо встановлювати CachyOS на SSD, а HDD використовуватимемо для зберігання даних.

| Диск | Розділ | Розмір | Файлова система | Точка монтування | Пояснення |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **SSD (238 ГБ)** | EFI | 2 ГБ | `fat32` | `/boot/efi` | Для завантажувача systemd-boot. Великий розмір — для драйверів NVIDIA. |
| | Root (`/`) | 70 ГБ | `btrfs` | `/` | Система та програми. btrfs дозволить робити снапшоти. |
| | Home (`/home`) | 166 ГБ (залишок) | `ext4` | `/home` | Ваші файли та налаштування. ext4 швидша для повсякденних завдань. |
| **HDD (1.82 ТБ)** | Data | весь диск | `ext4` | `/mnt/data` | Для архівів, медіа, резервних копій. |

## 3. Покрокове встановлення CachyOS

1.  Завантажтеся зі створеної флешки. (При появі чорного екрану додайте параметр `nomodeset` в GRUB).
2.  Запустіть інсталятор **"Launch Installer"**.
3.  На етапі вибору місця встановлення оберіть **"Ручне розділення" (Manual partitioning)**.
4.  **Створіть розділи на SSD:**
    *   `/dev/sda1`: 2 GiB, fat32, `/boot/efi`, прапорці `boot` та `esp`.
    *   `/dev/sda2`: 70 GiB, btrfs, `/`.
    *   `/dev/sda3`: залишок місця, ext4, `/home`.
5.  **Створіть розділ на HDD:**
    *   `/dev/sdb1`: весь простір, ext4, `/mnt/data`.
6.  **Налаштуйте завантажувач (Bootloader):**
    *   Оберіть **systemd-boot**.
    *   Вкажіть встановлення на диск `/dev/sda` (весь диск, а не розділ).
7.  Продовжте встановлення, оберіть робоче середовище **KDE Plasma** (рекомендовано для новачків).
8.  Після завершення встановлення перезавантажтеся та вийміть флешку.
